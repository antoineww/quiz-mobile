# Trivia Game Coding Challenge

## Overview

Your challenge is to create a 10 question, true or false, trivia app in React Native mobile.

## Available Scripts

In the project directory, you can run:

### To Install

```
yarn
```

### Start Server

```
yarn start
```

### Run Android

```
yarn android
```

### Run IOS

```
yarn ios
```
